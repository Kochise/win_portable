Сборка программы

Сборка программы осуществляется в среде Microsoft Visual Studio 2008 или 2010. Для разных версий VS в архиве присутствуют соответствующие файлы проектов.

ЧаВо (ЧАсто задаваемые ВОпросы)

В: Могу ли я использовать исходные коды программы в своей разработке с закрытым кодом?
О: Согласно лицензии Artistic License 2.0, код тех модулей, в которых используется опубликованный код, также должен быть опубликован. Под модулем понимается исполняемый файл или динамическая библиотека. Таким образом, вы можете использовать код программы в отдельной DLL-библиотеке, опубликовать код только этой библиотеки, а код основной программы оставить закрытым. Внедрять же опубликованный код непосредственно в закрытый модуль запрещено. Тем не менее, я не являюсь фанатиком идеологии open-source, и, если у вас есть существенные причины, по которым вы хотели бы использовать код для закрытого проекта, напишите мне — возможно, я пойду вам навстречу.
