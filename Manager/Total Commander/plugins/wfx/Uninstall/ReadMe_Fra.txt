UnInstaller v1.7.2 - Additif de Système de Fichiers pour Total Commander 5.51 et plus
----------------------------------------------------------------
Additif avancé pour désinstaller différents logiciels. 
Identique à celui du Panneau de configuration: "Ajouter\Supprimer des Programmes".
Plus simple d'emploi et plus puissant...

Capacitées:
------------
   - Montre toutes les entrées a désintaller (cachées également)
   
   - Désinstaller un program                       ("Entrée")
   - Voir les propriétées d'une entrée    ("F3" or "Ctrl-Q")
   - Supprimer des liens "morts"                ("Del" or "F8")
   - Editer des propriétées d'une entrée ("Alt"+"Entrée")
   - Configurer l'additif                ("Alt"+"Entrée" dans le "voisinage réseau" de TC)


Installation
------------
1. Dézipper l'archive dans un répertoire vide
2. Choisir Options->Configuration ->Opérations: Additif SF
3. Click "Ajouter"
4. Sélectionner UnInstTC.wfx dans le précédent répertoire
5. Click OK. Vous avez maintenant accés au plugin dans le "voisinage réseau"
