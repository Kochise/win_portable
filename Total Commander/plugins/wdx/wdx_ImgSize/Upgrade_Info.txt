ImgSize v1.08 (and higher), WDX Content Plugin for Total Commander, 2005 by J. Bleichroth

Upgrade from older Versions (1.07...) of this Plugin.


Because of the ImgSize Content Plugin Version 1.08 (and higher) internal rename of the english 
names of the columns 'x', 'y' and 'x * y' to 'Width', 'Height', 'Pixel' please "re-add" 
these columns in your TC Custom columns configuration:

- TC Menu -> Configuration -> Options -> Custom columns -> New/Edit ...
or
- Right-click on the (left or right) TC window columns - > Configure custom columns -> New/Edit ..



The section 'ImgSizeContentPlugin' in the file 'contplug.ini' is obsolete and can be deleted.



Thank you
