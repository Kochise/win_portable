UnInstaller v1.7.3 - FS-plugin Total Commander 5.51 vagy újabbhoz
-----------------------------------------------------------------

Fejlett plugin szoftverek eltávolításához.
Ugyanaz, mint a Vezérlõpult - "Programok hozzáadása/elvétele",
csak jobb és kényelmesebb.

Lehetõségek:
-------------
   - Minden program eltávolítási bejegyzésének
     megtekintése (rejtettek is) 
   - Program eltávolítása                     ("Enter")
   - Bejegyzés tulajdonságainak megtekintése  ("F3" vagy "Ctrl-Q")
   - Érvénytelen linkek törlése               ("Del" vagy "F8")
   - Bejegyzés tulajdonságainak szerkesztése  ("Alt"+"Enter")
   - Plugin konfigurálása                     ("Alt"+"Enter" a Hálózati Helyek-nél)


Telepítés:
-----------
1. Arhívum kitömörítése egy üres könyvtárba
2. Menübõl Konfiguráció - Beállítások - Mûködés - FS-beépülõk
3. Klikk "Hozzáad"
4. Keresd meg a kicsomagolt fájlokat, és jelöld ki a UnInstTC.wfx-t
5. Klikk OK. Mostantól elérhetõ a plugin a "Hálózati Helyek"-ben
