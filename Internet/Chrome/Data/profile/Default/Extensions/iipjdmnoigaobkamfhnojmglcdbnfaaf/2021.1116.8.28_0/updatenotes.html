ver 8.28.6

Supporter subscriptions are available again

Supporter subscriptions are now available again in the extension. Subscriptions are now managed with Stripe.

Supporter subscriptions encourage me to develop and maintain the extension. Supporters also get early or exclusive excess to several features, including domain blocklist, advanced url matching, tab count warnings, customisable keyboard shortcut actions, and more.

Supporter subscriptions were not available in the extension since February 2021 when Google ended in-app payment support. All existing supporters received free access to supporter features from February 2021, but new supporters were unable to sign up. This free access has now ended, and a new subscription is again required to access the supporter features.

Please sign-up for new subscriptions on the options screen.

Email address permission

Subscriptions require access to the email address with which you are signed-in to the browser. The subscription is linked to this email address, so it can be used on all computers where you are signed in to Chrome with the same email address.

ver 8.26.1

Updated: Added support for The Great Suspender alternatives

This update adds support for some extensions that are successors to The Great Suspender. The extensions currently supported are:

  1. The Marvellous Suspender
  2. The Great Suspender Original
  3. The Great Suspender

I personally use, and test the extension with The Marvellous Suspender. However, I have no association with, or recommend any of the above. This update is just to continue supporting suspending features with some of the successor extensions.

If you know another widely used, and compatible, successor to The Great Suspender, please do share it. I will try to add support for it.

ver 8.23.5

Announcement post for this update

NEW: Set a limit for the number of open tabs

Tab count limit available as a beta feature in the Advanced/Experimental settings section.

  • Soft limit—a nag notification appears on every tab when you exceed the tab limit.
  • Hard limit—no new tabs are allowed to be opened when you exceed the limit (not quite ready yet).
  • Tab count background colour changes when approaching/exceeding limit
    • red if over limit
    • amber if close to limit
    • green if well below limit

Tab count limits will be a supporter-only feature when it is ready, but is available to all while still under beta.
