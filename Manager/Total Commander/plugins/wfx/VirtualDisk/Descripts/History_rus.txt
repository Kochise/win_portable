Virtual Disk, Версия 1.3.3 Final
  FS-плагин для программы Total Commander, позволяющий подключать образы дисков
  в качестве дополнительных виртуальных дисков в системе.

Автор плагина: Власов Константин, 2012 г.
Домашняя страница: http://flint-inc.ru/
E-mail:            support@flint_inc.ru

Драйвер основан на open-source проекте FileDisk: Bo Branten, 2009 г.
Домашняя страница: http://www.acc.umu.se/~bosse/
E-mail:            bosse@acc.umu.se


    /===========================\
    |      История версий:      |
    \===========================/

Версия 1.3.3 Final (build 5300):
  1. Драйвер теперь подписан цифровой подписью Кристиана Гислера, что позволяет
     использовать его в Windows Vista/7 x64 без отключения проверки подписей.
  2. Исправлена уязвимость: при помощи драйвера пользователи могли работать с
     файлами, доступ к которым был для них запрещён (исправление реализовано
     только для WinXP и новее!).
  3. Запрещено подключать сжатые и зашифрованные образы: из-за ошибки Windows
     это может приводить к зависанию драйвера и системы.
  4. Добавлен китайский (традиционный) перевод интерфейса (Sheree Chen).
  5. В диалоге свойств образа фокус по умолчанию теперь установлен на кнопке
     подключения/отключения.
  6. Реализована утилита командной строки vd_filedisk.exe для управления
     подключением образов независимо от TC (загружается отдельно).
  7. Выложены исходные коды драйвера и утилиты vd_filedisk.exe по лицензии GPL.
  8. Добавлена 64-битная версия.

Версия 1.3.2 (только драйвер):
  1. Обновлён драйвер в соответствии с изменениями, внесёнными в оригинальный
     FileDisk 1.17.0.0:
    а) исправлена проблема с нераспознающимися ISO-образами с файловой системой
       Joliet под Windows Vista/7.

Версия 1.31 Final (build 4300):
  1. Обновлён драйвер в соответствии с изменениями, внесёнными в оригинальный
     FileDisk 1.15.0.0:
    а) добавлена поддержка CD/DVD-дисков в формате UDF;
    б) улучшена поддержка подключения образов из сети;
    в) мелкие исправления и улучшения.

Версия 1.30 Final (build 3001):
  1. Несколько улучшена система нумерации версий.
  2. Добавлены сообщения, более подробно описывающие некоторые ошибки.
  3. В ReadMe-файлы добавлена секция ЧаВо.
  4. Добавлен венгерский перевод интерфейса (Robert Bukovinszki).
  5. Добавлен полный итальянский перевод (Paolo Gai).
  6. Добавлен чешский перевод интерфейса (Martin Ruzicka).
  7. Добавлен турецкий перевод интерфейса и файла ReadMe (Atila Ozkok).

Версия 1.30 beta 3:
  1. Реализовано автоподключение образов при перезагрузке.
  2. Исправлена ошибка, из-за которой диски, отключённые в результате перезагрузки,
     отображались подключёнными.
  3. Добавлены 64-битные драйверы: протестированный для Windows x64 и пробный для
     Windows 64-bit (Intel Itanium).
  4. Добавлены два файла немецкого перевода (авторы: Frank Bultge и Peter Brockamp).
  5. Несколько мелких исправлений.

Версия 1.30 beta 2:
  1. Драйвер изменён так, что теперь для работы с плагином не требуются права администратора. Они
     необходимы только для установки драйвера.
  2. К новому способу установки драйвера (через INF-файл) добавлен старый (через ручное копирование
     файла и импорт данных в реестр), т.к. оказалось, что новый способ подходит не для всех.
  3. Файлы NRG теперь добавляются по умолчанию в режиме CD/DVD, т.к. этот формат в некоторых случаях
     подключается и распознаётся корректно.
  4. В качестве эксперимента добавлена 64-битная версия драйвера VD_FileDisk.

Версия 1.30 beta 1:
  1. Добавлен режим эмуляции флоппи-дисков.
  2. Драйвер переименован из FileDisk в VD_FileDisk, чтобы исключить конфликты с другими программами,
     использующими похожий драйвер (например, WinImage 7.0+).
  3. Упрощена установка драйвера (теперь она выполняется при помощи INF-файла).
  4. Добавлен польский перевод интерфейса (Pawel Wawrzyszko).
  5. Добавлен украинский перевод интерфейса (Butsky Igor).

Версия 1.26:
  1. Исправлена ошибка, из-за которой название группы переключателей "Режим подключения" не переводилось
     при смене языка.
  2. Изменён шрифт диалога с MS Sans Serif на "виртуальный" MS Shell Dlg, чтобы его можно было
     самостоятельно изменять на каждой системе.
  3. Добавлены французский (Claude Charries) и испанский (Luis Mejia) переводы.

Версия 1.25:
  1. Плагин теперь поддерживает мультиязычность при помощи LNG-файлов. Язык можно выбрать,
     вызвав диалог "О программе" (пункт "Свойства" в контекстном меню, или нажатие Alt+Enter
     на пункте "Virtual Disks") и нажав кнопку "Настройки". В комплекте с плагином идут два
     языка: русский и английский, второй выбран по умолчанию.
  2. В диалоги добавлены клавиши для быстрого перехода между элементами управления (подчёркнутые
     символы).
  3. Из комплекта убрана программа Convert.exe как неактуальная.
  4. Добавлен файл pluginst.inf для поддержки автоматической установки в Total Commander 6.5+.

Версия 1.20:
  1. В этой версии используется обновлённый драйвер filedisk.sys от Bo Branten'а (версия 1.0.0.13).
     В нём исправлена ошибка, приводившая к утечкам памяти, в результате которых при работе с
     виртуальным диском система работала всё медленнее вплоть до полного зависания.
     Также в драйвере улучшена работа с большими образами CD (более 700 МБ).

Версия 1.10:
  1. Весь код плагина полностью переписан. В результате работа со списком файлов стала
     более оптимизированной, а также намного улучшена работа одновременно из нескольких
     копий TC.
  2. Уравнены в правах расширения ISO и BIN, т.к. выяснилось, что оба они распознаются
     Windows в ккачестве виртуальных дисков (иконки, тип подключения...).
  3. Добавлено автоопределение типа подключения при добавлении файла.
  4. Исправлено несколько багов при добавлении/удалении файлов.
  5. Запрещено удалять подключённые образы, не отключив их.

Версия 1.00:
  Первая версия. Как оказалось, довольно глючная. :) Умеет подключать диски ISO, BIN (режим CD)
  и IMG (режимы RW и RO).
