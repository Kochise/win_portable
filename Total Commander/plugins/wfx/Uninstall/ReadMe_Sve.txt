UnInstaller v1.7.3 - FS-plugin för Total Commander 5.51 och högre
-----------------------------------------------------------------

Insticksmodul för att avinstallera program. Samma som
Kontrollpanelen - "Lägg till\ta bort program", men bekvämare och med fler funktioner.

Funktioner:
------------
   - Show all records for uninstall program 
     (även dolda) 
   - Avinstallera program                     ("Enter")
   - Visa postens alla egenskaper             ("F3" eller "Ctrl-Q")
   - Ta bort felaktiga länkar                 ("Del" eller "F8")
   - Redigera egenskaper för post             ("Alt"+"Enter")
   - Konfigurera insticksmodulen (plugin)     ("Alt"+"Enter" i Nätverk)


Installation
------------
1. Zippa upp arkivet i en tom mapp
2. Välj Konfigurera - Generella inställningar - Funktioner - FS-plugins
3. Välj "Lägg till"
4. Gå till mappen där du zippade upp arkivet och välj UnInstTC.wfx
5. Klicka på OK. Nu kommer du åt denna insticksmodul (plugin) i "Nätverk"
