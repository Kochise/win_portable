Name

miktex-links — manage links from scripts and formats to executables

Synopsis

miktex [common-option...] links command [command-option-or-parameter...]

Description

Commands for managing links from scripts and formats to executables.

Commands

install [--force]

Install all links. Existing links will be overwritten, if --force is specified.

list [--template template]

List all links.

template controls the output of each record. It can contain the following placeholders:

{linkName}, {linkType}, {target}

uninstall

Uninstall all links.

See also

miktex(1)
