Plugin 'Calendar' pour Total Commander

Fabio Chelly,
fabio@chelly.net

Site Internet:	  http://totalcommander.free.fr/fabio
Téléchargement:   http://totalcommander.free.fr/fabio/calendar.zip


Calendar est un plugin qui calcule et affiche des informations
sur les phases de la lune et du soleil, différents calendriers,
des fêtes et évènements.
Il permet en outre de synchroniser l'horloge de son PC avec l'heure
atomique disponible sur Internet sur certains serveurs.

______________________________

        INSTALLATION
______________________________

1. Décompressez le contenu du fichier ZIP dans le dossier de votre
   choix. Je vous conseille de choisir un sous-dossier du répertoire
   où est installé Total Commander:
   Par exemple, le dossier C:\totalcmd\plugins\WFX\Calendar\

2. Allez dans le menu      : Options > Configuration
   Selectionnez l'onglet   : Opération
   Cliquez sur le bouton   : S.F.-Additifs
   Cliquez sur le bouton   : Ajouter
   Choisissez le fichier   : calendar.wfx (dans le dossier où a été
       			     décompressé le ZIP)
   Cliquez sur OK (3 fois)

3. Le plugin 'Calendar' est désormais installé.


______________________________

    COMMENT L'UTILISER
______________________________

1. Sous Total Commander, allez dans le voisinage réseau
2. Double cliquez sur 'Calendar'
3. Parcourez la liste des options pour obtenir l'information qui vous intéresse.

______________________________

            NOTES
______________________________

- Les dates et heures sont affichées comme date et heures des fichiers
- Je vous conseille de classer la fenêtre par 'Taille' croissante pour
  que les informations soient plus lisibles
