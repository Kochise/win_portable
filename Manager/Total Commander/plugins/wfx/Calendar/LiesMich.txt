Kalender ist ein Plugin für Total Commander. Es zeigt Information
über Sonnen-, Monddaten und Feiertage an. Sie können zudem noch die
Uhr ihres Computers mit einer Atomuhr via Internet synchronisieren.


______________________________

        INSTALLATION
______________________________

1. Kopieren Sie den Inhalt des Archives "calendar.zip" in einen Ordner
   ihrer Wahl. Es wird empfohlen das Plugin in ein Unterverzeichnis vom Total
   Commander Ordner zu installieren. z.B. C:\totalcmd\plugins\WFX\Calendar\

2. In der Menüleiste        : Konfigurieren > Einstellungen
   Wählen Sie den Reiter    : Operation
   Klick Sie auf den Button : DS-Plugins (ein Dialog erscheint)
   Klick Sie auf den Button : Hinzufügen
   Wählen Sie die Datei     : calendar.wfx (vom Ordner in welchem Sie
                              das Archiv zuvor entpackt haben)
   Klicken Sie auf OK (3 mal)

3. Das Kalender Plugin ist jetzt installiert.


______________________________

        WIE BENUTZT MAN ES
______________________________

1. Im Total Commander 'Netzwerkumgebung' auswählen
2. Doppelklick auf 'Calendar'
3. Sie könnne nun verschienden Einstellungen vornehmen und Informationen entnehmen.

______________________________

            HINWEISE
______________________________

- Das Datum und die Zeit werden als Dateizeit/datum dargestellt
- Es wird empfohlen die Liste nach Größe aufsteigend zu ordnen.
