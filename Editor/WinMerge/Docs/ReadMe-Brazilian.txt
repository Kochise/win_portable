WINMERGE

WinMerge é um utilitário de comparação e união de arquivos com o código fonte aberto o qual roda em todas as versões modernas do windows. Algumas funções requerem instalações ou componentes adicionais. A última versão do WinMerge e outras informações do WinMerge estão disponíveis em 
https://winmerge.org

 Início rápido para o WinMerge: Leia o capítulo do início rápido do manual online para se iniciar com o WinMerge:
 https://manual.winmerge.org/QuickStart.html

 Manual em HTML: O manual está disponível online em 
https://manual.winmerge.org/ ele está também possivelmente instalado (se escolhido assim) localmente e descarregável separadamente de https://winmerge.org/ (veja a documentação) 

 Suporte a Scripts:
 Se você gostaria de trabalhar com scripts Você precisará ter o Windows 
Scripting Host instalado. Se você receber quaisquer erros relacionados aos seus 
scripts por favor visite 
https://msdn.microsoft.com/library/default.asp?url=/downloads/list/webdev.asp
 para ter certeza que seu Scripting Host está ambos atualizado e não corrompido.
