ICLView.wlx
v25.9.2005
Free for noncommercial use.

Плагин для Total Commander. Показывает иконки в файлах во встроеном просмотрщике.
Поддерживаемые расширения: ico, icl, exe, dll, scr, ocx, bpl, wlx, wfx, wcx, wdx, cpl, acm.

ВНИМАНИЕ! Удалите предыдущую версию (если установлена) из списка плагинов просмотра (это даст возможность после установки новой версии смотреть иконки (ico-файлы) в TC6 и выше). Также добавлены новые типы файлов.


Установка (TC 5.51):

1. Если предыдущая версия плагина установлена, удалите ее из списка установленных плагинов.
2. В настройках на вкладке просмотра/редактирования нажмите кнопку настройки внутреннего просмотровщика.
3. В появившемся окне нажмите кнопку LS-Plugins.
4. В появившемся окне нажмите кнопку Добавить и выберите ICLView.wlx.
5. Закройте окна нажав "ОК".
Плагин установлен.


Small FAQ.

Q. What the items who contains zero icons?
A. This compressed icon (cannot be extrtacted) or icon who contain only valid header without image (can be extracted).
Q. I can't see icons in exe, dll files. What can I do?
A. If started other plugin first then try press button 4 in Lister.
Q. Can I open Stardock Icon Packager icon files (*.ip)?
A. Stardock Icon Packager icon files (*.ip) is zip archives contained *.icl file inside. Enter to *.ip file by pressing Ctrl-PgDown in TC and see (by F3) *.icl file. (Answer by Wuyko WINDOWS).
